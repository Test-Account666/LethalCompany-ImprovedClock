# ImprovedClock

Your clock, but on steroids!

Allows you to adjust some aspects of the clock.

Current features:

- Clock Recolor
- Realtime Clock
- 24-Hour Clock
- Spectator Clock
- Clock Visibility in Ship
- Clock Visibility in Facility
